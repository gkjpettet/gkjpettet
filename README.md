# Garry Pettet

Hi, I’m Garry — a consultant radiologist and indie software developer.

I build software for people who like tools they can understand: visual development environments, small languages, desktop apps, developer utilities, and the occasional odd-but-useful side project.

These days all of my development energy goes into [Objo Studio](https://objo.dev), a friendly, visual cross-platform environment for building desktop apps.

## Objo Studio

[Objo Studio](https://objo.dev) is my attempt to make desktop app development feel approachable again.

It combines:

- a visual user interface designer
- a readable, BASIC-inspired language
- a stack-based virtual machine
- a modern .NET/Avalonia app runtime
- integrated debugging and project tooling
- a focus on hobbyists, learners, indie developers, and people who miss the immediacy of classic VB-like tools

Objo Studio is not trying to be another heavyweight professional stack. It is for people who want to build real apps without first adopting the entire modern software industry as a lifestyle.
